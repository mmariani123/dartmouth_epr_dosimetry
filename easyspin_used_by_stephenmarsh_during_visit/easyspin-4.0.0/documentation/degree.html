degree
Synopsis

Conversion from radians to degrees 

value = degree
Description

degree returns the conversion factor between radians and degrees.

EasySpin functions only accept angles in radians. To convert an angle from degrees to radians, multiply by degree: 

phi = 60*degree;

On the other hand, to convert an angle from radians to degrees, divide by degree: 

phi = pi/7;          % angle in radians
phi_d = phi/degree;  % gives angle in degrees
Reference

degree returns a simple number: pi/180 = 0.017453292519943
