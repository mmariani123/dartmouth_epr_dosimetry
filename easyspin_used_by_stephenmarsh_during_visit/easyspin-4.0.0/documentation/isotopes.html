isotopes
Description

This graphical interface provides an interactive PSE (periodic system of element) with a list of nuclear isotope data.

The columns in the table below the PSE contain the following data for each nuclear isotope

  1. Mass number
  2. Element symbol
  3. Nuclear spin, multiple of 1/2
  4. gn value, including sign
  5. ENDOR/NMR frequency (MHz) for the specified magnetic field
  6. Nuclear electric quadrupole moment (in barn)
  7. Natural abundance in percent

EasySpin's nuclear isotope database includes all stable isotopes plus the most important radioactive ones.
