mhz2mt
Synopsis

Conversion from MHz (Megahertz) to mT (Millitesla). 

B = mhz2mt(nu)
B = mhz2mt(nu,g)
Description

mhz2mt takes the frequency value of nu, assumes that it is in units of MHz (Megahertz), converts it to magnetic field in units of mT (Millitesla) and returns the result in B.

g gives the g factor for the conversion. If it is omitted, the g factor of the free electron 2.0023... is used (see gfree).

nu can be a vector of values. In this case, g can be a scalar or a vector of the same size as nu.
