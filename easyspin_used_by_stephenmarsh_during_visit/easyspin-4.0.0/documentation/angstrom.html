angstrom
Synopsis

Conventional unit for distances on the molecular scale. 

value = angstrom
Description

angstrom returns the SI value (in meters) of 1 Angstrom in value.

Reference

1 Angstrom = 100 pm = 0.1 nm = 10-10 m.
